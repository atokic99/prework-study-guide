HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Opening and closing tags are represented with angle brackets.
  • HTML attributes are always defined in the beginning element, never the closing element.
  • The title element defines the title for for a webpage.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadata such as the character set, descriptions, keywords, author, and viewpoints.
  • The charset element sets our character encoding.
  • The viewport element sets the default view for the webpage.
  • The http-equiv element sets an HTTP header for content.
  • Elements inside other elements are called child elements. The element that holds all the kids are called parent elements, parent/child language is used the explain relationships among elements in HTML.
  • The body element is an example of a semantic HTML element.
  • div creates division in the page or span to create an inline division.
  • The p element represents a paragraph or block of text.
  • The ul, ol, and li, elements represent unordered lists, ordered lists, and list items.
  • The ul element contains an unordered list or bullet list.
  • The img element contains info about the images that are displayed on the webpage.
  • Cont. the src attribute is the most important because it defines the location of the image.
  • The br element creates a line of empty space, or a line break of content.
  • The a element (which stands for anchor) creates links to the same webpage or other webpages.
  • The herf attribute points to the URL for the link.

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • To style an element directly in the HTML file, they use inline CSS.
  • The period before card makes this a class selector.
  • Any HTMl element that is assigned to the class, card, will get the property assigned from CSS.
  • The box-shadow CSS property assds shadow effects around an element's frame.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

JavaScript

  • To declare a variable use keyword var.
  • After the var keyword, we give the variable a name.
  • Use an assignment operator = to assign the specific data stored in our varible.
  • Console refers to a test environment that developers use to check out their code.
  • .log() is a method, is a method, which means a set of instructions that can be executed by a computer.
  • The .log() method will output whatever we add inside the parentheses to the console.
  • To link JavaScript to HTML we use similar steps as we did to CSS, but we add it under the footer element in HTML.
  • Control Flow is the order in which a computer executes code in a script.
  • JavaScript is read top to bottom, and order counts. Must be intentional about how we set up code in a JS file; can't just throw all in there in any way!
  • Control Flow can be interuppted, but we can use conditional statments and loops to control the order in which code is executed.
  • An if statment is a way of introducing decision-making into our code.
  • The if statment code, will only execute if the statment meets a condition, or is true.
  • When the computer encounters an if statement in a JavaScript file, it will check to see if the condition that we provide is truthy.
  • If the "if" statment provides to be truthy, then it will do whatever action we ask of it to do in between the curly brakets i.e."{}".
  • But if the condition isn't true or have a value that isn't truthy, then the computer will ignore whatever is in between the culry brackets and move on.
  • An array is a single variable that is used to hold a group of data.
  • Arrays are useful tools for storing multiple items with different data types in a single variable, but not all arrays must have multiple data types.
  • To work with individual data items inside an array, we need to have one more piece of information: the location of the data item in the array.
  • This location is identified with a unique number called an index.
  • In JavaScript and many other programming languages, arrays are zero-indexed and sequential. Meaning that the first item in any array has an index of 0 not 1.
  • A "for" loop uses the perdictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over again.
  • By using a "for" loop, we can tell the computer to go through each og the shapes in our array and log them to the console.
  • The array length is important because the for loop needs to know how many times the code will run until there are no items left in the array.
  • We can find the length of an array by using a built-in property of an array, appropriatly named "length".
  • A "for" loop contains 3 important statements.
  • The first statment determins the stating point for our loop.
  • The second statment is the condition.
  • The third statment is what allows the array to iterate over each item.
